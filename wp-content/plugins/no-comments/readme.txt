== Description ==

No Comments plugin totally get's rid of comments. Just activate the plugin and it's all gone! No settings, just activate.

== Installation ==

1. Upload `no-comments` folder to the `/wp-content/plugins/` directory or upload no-comments.zip via wordpress upload button in the plugins area in the backend.
2. Activate the plugin through the 'Plugins' menu in WordPress
